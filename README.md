# Fortune Construction 🏗️

Welcome to the Fortune Construction landing page. This is a premium, modern web application built to showcase high-end construction, infrastructure, and architectural projects.

## ✨ Features

- **Dynamic Video Slider Hero**: An immersive hero section that smoothly automates crossfading transitions across multiple high-quality video backgrounds.
- **Premium Aesthetics**: Features a sophisticated dark theme (`#0F1115`) with elegant gold accents (`#C5A059`), glassmorphism panels, and elegant typography (Playfair Display & Inter).
- **Interactive Framer Motion Components**: 
  - An animated "GET A QUOTE" call-to-action button with a continuous glowing heartbeat pulse, scaling effects, and tactile tap feedback.
  - Smooth page scrolling for navigation.
  - Scroll-based reveal animations across different sections (Impact Tracker, Expertise).
- **AI Integration (Fortune AI Lab)**: A mock interface showcasing how AI can be utilized to generate blueprints, analyze site photos, and render architectural video concepts.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Motion (Framer Motion)](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: TypeScript

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Environment Setup** (Optional for AI features)
   Create a `.env.local` file in the root directory and add your Gemini API Key if you want to test the AI Lab integrations:
   ```env
   GEMINI_API_KEY="your_api_key_here"
   ```

3. **Run the Development Server**
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000/` in your browser to view the application.

## 📦 Building for Production

To create a production-ready build:
```bash
npm run build
```
This command compiles and optimizes the React application via Vite.
